Nearest  Neighbor

Analytos's nearest neighbor toolkit provides the most flexible way to search for data points that are similar to a set of queries,
does so at interactive speed.

 

Why Use a nearest neighbor / knn algorithm?

Finding the nearest neighbors of a set of query data points is a core component of many machine learning algorithms. The nearest neighbor classifier, for example, predicts the label for a query point based on the labels of the closest points in the training set.

 

The Analytos Difference  :  Nearest Neighbors

Nearest neighbors search tools usually trade flexibility and usability in the interest of speed; the Analytos toolkit offers both. "Composite distance" functions allow comparisons between data points with any mix of data types, like numeric values and free text. Expert users can choose from one of several search strategies, or the toolkit can choose the best option based on characteristics of the data.

 

The nearest neighbors toolkit runs on top of Analytos's highly-efficient out-of-core data structures, which enables searching within extremely large reference datasets at interactive speeds. Finally, Analytos's nearest neighbors models can be saved and deployed like any other machine learning model, so they can be used for real-time search querying.
