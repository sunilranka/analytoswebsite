Logistic  Regression

Logistic regression is a model that is popularly used for classification tasks.
Analytos’s classification toolkit provides a complete set of features from creation of a model over predicting outcomes to evaluating the accuracy of the model.

 

Why Use a Logistic Regression?

Logistic regression is used whenever you need to assign an input to one of several classes. It applies a logistic function to a linear combination of features, yielding a probability for an input to be in one of the classes. The output is usually binary, but logistic regression can also be applied to multiclass classification problems.


The Analytos Difference  :  Logistic Regression

logistic regression toolkit offers four advantages:

  • It works on all kinds of data, from dense to sparse.
  • It automatically picks the right solver based on your data types; no manual tuning is required.
  • It is heavily optimized for speed.
  • It transparently scales out to multple machines when used in a distributed execution context.

 

For binary classification, Analytos's logistic regression API provides intuitive interfaces for the three types of predictions: The class of the input, the probability estimate for being in either class, and the margin to the model’s linear decision boundary. In addition to binary classification, logistic regression models can be trained for multiclass labels. The prediction output can then provide the top-k class predictions for each sample.

 

A model can be easily evaluated by providing APIs for the classification accuracy (the fraction of test inputs with correct predictions) and confusion matrix (comparing the numbers of correct and incorrect predictions).

 

Moreover, If the data is imbalanced because one class is underrepresented in the training data, the model can compensate for the imbalance by applying asymmetric costs of misclassifying elements.
